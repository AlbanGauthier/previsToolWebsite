The Project

INTRODUCTION & MOTIVATION

This project explores a number of solutions to be used by small film productions in order to integrate state-of-the-art interaction experiences using Augmented Reality.

GOALS

This project aims to provide small movie production with a collaborative tool which would allow them to work with animated storyboards. They are now able to edit complex scenes in real time using their phones and a PC, and then shoot a storyboard using the PC's Graphics Card.

GRAPHICS

We explore various techniques such as real time light casting and shadows in Augmented Reality. We also use current generation PC techniques such as Screen Space Reflection, Depth of Field, Chromatic Aberrations and many more on the PC Version of the Project to be able to get the effects movie productions are looking for.

INTERACTION

The project aims at creating a new experience of user interaction by exploring brand new Augmented Reality libraries for Android and iPhone. The smartphone view allows to place, move, scale and rotate objects, as well as be used as a ingame camera.

LEARNING

We learned to work in an international team (french, spanish, indonesian and swedish) to use existing AR technologies in Unity, and create complex real time scenes, using assets which can be rendered in real time on both smartphones and PC. Some networking skills were necessary to create a real time multiplayer experience.
